# 🧮 Java Calculator Project (Step 01)
이 프로젝트는 자바 기초를 다지기 위한 계산기 로직 구현 프로젝트입니다.

단순한 계산기 구현을 넘어, JUnit과 같은 테스트 프레임워크의 동작 원리를 이해하기 위해 직접 Assertion 로직과 빌드 스크립트를 구현한 프로젝트입니다.


## 🚩 프로젝트 핵심 목표
1. **바닥부터 시작하는 빌드 환경**: IDE의 도움 없이 CLI(`javac`, `java`)와 배추 파일(`.bat`)을 통해 직접 빌드 파이프라인 구축
2. **테스트 프레임워크의 본질 체득**: 가드 절(Guard Clause)을 활용한 `expect` 함수 설계 및 검증 자동화
3. **안정성 확보**: 정수 오버플로우 등 경계값 테스트를 통한 로직 검증


## 🛠️ 기술적 의사결정 (Why?)

### 1. `build.bat`을 통한 Clean Build
- 이전 빌드의 잔여 `.class` 파일이 현재 테스트에 영향을 주지 않도록 매번 `bin` 폴더를 초기화하고 컴파일하는 방식을 채택
- 인코딩 이슈를 방지하기 위해 `-encoding UTF-8` 옵션을 명시

### 2. 객체지향적 테스트 설계
- `static` 호출 대신 `new Calculator()`를 통해 인스턴스를 생성하여 사용함으로써 자바의 객체지향 원칙을 준수
